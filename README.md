# Titanic Survival Prediction

## Objective
To predict whether a passenger survived the Titanic disaster using machine learning.

## Dataset
Titanic Dataset from Kaggle

## Models Used
- Logistic Regression
- Random Forest

## Best Accuracy
92.73% (Random forest after hyperparemeter tuning)
